What Is Node.js

  • A server-side JavaScript platform based on Chrome V8
  • Single-threaded, event-driven programming environment
  • Performing Non-blocking(Asynchronous) I/O

What's It For

  • It's Goal is to provide an easy way to build scalable network programs
  • Can be used as:
    • Web Server
    • TCP Server
    • Proxy Server
    • Anything that has to deal with high I/O

Already so many technologies out there, Why Node.js

  • Asynchronous I/O is better for high I/O application
  • Single-threaded and event-driven is better than multi-threaded
  • Based on Chrom V8, Fast!
  • Lightweight, easy to learn
  • One language for front-end and back-end
  • Cross platform, Mac, Linux, Windows
  • Open Source, MIT license
  • Active community, over 25, 000 modules

Quick Start

Install

  • http://nodejs.org/download/, select a proper version

A Basic Web Server

  • webServer.js: 
    var http = require('http');
http.createServer(function (req, res) {
    res.end('Hello world!');
}).listen(3000);
  • node webServer.js

Web Server With Routing

  • npm install express
  • webServer2.js: 
    var express = require('express');
    var app = express();
    app.get('/hello', function (req, res) {
        res.send('world');
    });
    app.get('/hi', function (req, res) {
        res.send('Node.js');
    });
    app.listen(3000);
  • node webServer2.js

NPM

  • Node.js Package Manager
  • https://npmjs.org/
  • Install a package: npm install package-name
  • Find a package in NPM repository: npm search package-name
  • Publish a package to NPM repository: npm publish tarball/folder
  • List packages: npm ls
  • Most starred packages: express, request, async, socket.io, mocha, underscore, grunt, redis, jade, connect...
