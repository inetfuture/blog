What Is Node.js

  • A server-side JavaScript platform based on Chrome V8
  • Event-driven, single-threaded, programming environment
  • Performing Non-blocking(Asynchronous) I/O
  • Not a library, framework or new language

What's It For

  • It's Goal is to provide an easy way to build scalable network programs
  • Primarily be used as:
    • Web Server
    • TCP Server
    • Proxy Server
    • Anything that has to deal with high I/O
  • Also:
    • Command Line Tool

Already so many technologies out there, Why Node.js

  • Asynchronous I/O is better for high I/O application
  • Event-driven and single-threaded is faster than multi-threaded under high concurrency
  • Based on Chrom V8, Fast!
  • Lightweight, easy to learn, concise api
  • A good balance between performance and productivity
  • Cross platform, Linux, Mac, SunOS, Windows
  • Open Source, MIT license
  • Active community, over 25, 000 modules in less than 5 years
  • JavaScript!!! One language for front-end and back-end

Quick Start

Install

  • http://nodejs.org/download/, select a proper binary version

A Basic Web Server

  • webServer.js: 
    var http = require('http');
http.createServer(function (req, res) {
    res.end('Hello world!');
}).listen(3000);
  • node webServer.js

Web Server With Routing

  • npm install express
  • webServer2.js: 
    var express = require('express');
var app = express();
app.get('/hello', function (req, res) {
    res.send('world');
});
app.get('/hi', function (req, res) {
    res.send('Node.js');
});
app.listen(3000);
  • node webServer2.js

RealTime App Demos

  • Supported transports:

Meteor - A better way to build apps

  • Clean, powerful data synchronization
    Write your client code as if it were running on the server and had direct access to the database. No more loading your data from REST endpoints.
  • Live page updates
    Just write your templates. They automatically update when data in the database changes. No more boilerplate redraw code to write.
  • Based on Node.js and MongoDB.
  • Demo: Mini trello

    • More analogous frameworks

    Derby, SocketStream... all based on Node.js
